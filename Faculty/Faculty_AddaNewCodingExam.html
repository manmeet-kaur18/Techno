Instructions to How to Create Coding Questions
Question : Enter the question specifying all the constraint requirements and the inputs that code will require to execute
A certain pattern needs to be followed to give the testcases as input to schedule coding exam. So follow the below instructions to input the testcase -
Sample Testcase/ Testcase : The input needs to follow the below given pattern while entering it
[[testcase1],[testcase2],[testcase3]]
Now the testcase will be a comma seperated list of inputs required for that particular testcase
For Example : [[1,2,3],[3,4,5],[4,5,6]] here 1,2,3 is inputs for testcase1, 3,4,5 are inputs for testcase 2 and so on
Now for single sample testcase/ testcase format is [[1,2,3]] - where 1,2,3 is inputs for testcase
